# Breakout Game (벽돌깨기)

HTML5 Canvas를 사용한 고전 벽돌깨기 게임입니다.

## 게임 방법

1. **게임 시작** 버튼을 클릭하여 게임을 시작합니다.
2. 화살표 키(← →) 또는 마우스를 움직여 하단의 패들을 조작합니다.
3. 공을 튕겨서 위의 벽돌들을 모두 깨세요!
4. 공이 바닥에 떨어지면 생명을 잃습니다. (총 3개의 생명)
5. 모든 벽돌을 깨면 승리합니다!

## 특징

- 6개 줄, 10개 열의 다채로운 벽돌
- 화살표 키 또는 마우스 컨트롤
- 점수 및 생명 표시
- 일시정지 및 다시하기 기능
- 반응형 디자인

## 실행 방법

`index.html` 파일을 웹 브라우저에서 열면 바로 플레이할 수 있습니다.

## 파일 구조

- `index.html` - 게임 메인 페이지
- `style.css` - 스타일시트
- `game.js` - 게임 로직

## 기술 스택

- HTML5 Canvas
- CSS3
- Vanilla JavaScript

## 조작법

- **키보드**: 좌우 화살표 키
- **마우스**: 캔버스 위에서 마우스를 좌우로 움직임
- **게임 시작**: 시작 버튼 클릭
- **일시정지/재개**: 일시정지 버튼 클릭
- **다시하기**: 다시하기 버튼 클릭

즐거운 게임 되세요! 🎮
